# Lab Environment Setup

This project was performed using a small and controlled
wireless lab environment to simulate real-world scenarios.

## Lab Components

- Attacker machine: Kali Linux
- Wireless adapter with monitor mode support
- Test Wi-Fi network using WPA2-PSK
- One connected client device

## Purpose of the Lab

The goal of this setup is to replicate a common home or
small office wireless network, which is often targeted
during wireless attacks.
