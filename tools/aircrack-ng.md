# Aircrack-ng Overview

Aircrack-ng is a widely used suite of tools designed
for wireless network security testing.

## Usage in This Project

In this project, Aircrack-ng was used to:
- Monitor wireless traffic
- Capture WPA/WPA2 handshakes
- Analyze wireless network security

### Aircrack-ng is commonly used by penetration testers to assess the strength of Wi-Fi security configurations.
